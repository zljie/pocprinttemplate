# 平台任务拆分

## 数据源管理模块
- 列表页｜展示数据源名称、模块、来源、更新时间、最后编辑人、状态；支持分页
- 搜索功能｜按编码或名称关键字过滤列表，结果实时更新
- 新增按钮｜点击进入新增表单页，完成创建后返回列表并显示新数据源
- 新增表单｜填写名称、模块、来源、参数对象选择与排序；保存成功持久化并校验必填
- 编辑页面｜可修改基础信息与已选参数对象别名、是否集合、顺序；保存后更新列表
- 删除功能｜列表行支持删除，二次确认，删除后列表移除并持久化

## 打印模板管理模块
- 列表页｜展示模板名称、编码、模块、数据源名称、纸张设置、状态；支持分页
- 搜索功能｜按名称或编码过滤列表，输入实时生效
- 新增按钮｜进入新建模板表单，含纸张边距预设与自定义数值
- 新增表单｜完成模板基础信息保存，创建后进入设计页面
- 编辑页面｜进入 TinyMCE 编辑器，支持插入占位符与实时预览
- 删除功能｜支持删除模板，二次确认后移除并持久化

## 模板设计与预览
- 编辑器加载｜TinyMCE 本地兜底加载成功，初始化模板内容
- 字段插入｜右侧字段选择支持 header/detail/summary 插入占位符到光标处
- 预览数据弹窗｜点击预览弹出 JSON 输入框，默认示例数据，校验后写入会话
- 模板+数据预览｜注入 previewTemplate+previewData，跳转案例页统一渲染
- 打印按钮｜预览页提供打印按钮，触发浏览器打印对话框
